# Project
### 배틀그라운드 e스포츠 데이터 분석
* 프로젝트 기간 : 2023.04.27 - 2023.05.09
* 프로젝트 주제 : 배틀그라운드 프로 선수 경기 데이터 분석 및 2023 PGS1 우승팀 예측  

  1-1) 그룹 스테이지 만의 데이터로 우승권 예측해보기  
  
  1-2) 우승을 예측하면서 발견한 데이터, 정보를 바탕으로 프로 선수 및 신규 유저에게 정보 제공  
  
  1-3) 프로젝트에서 예측한 우승권과 실제 우승권을 비교
